# System Logger Plugin

This plugin provides a system logger tab in the bottom panel of the application.

## Features

- **Real-time Logs**: Displays system logs as they occur.
- **Auto-scroll**: Automatically scrolls to the latest log entry.
- **Clear Logs**: Button to clear the current log history.
- **Bottom Panel Integration**: seamlessly integrates into the AlephNet bottom panel.

## Development

The source code is in `renderer/index.tsx` and `main/index.ts`.
To build the plugin, run:

```bash
node scripts/build-plugins.js
```
